## How To RUn

1. Clone this repository
   ```
   $ git clone https://github.com/agusnuryady/InventoriAppFrontEnd.git
   ```
2. Install all depedencies on the package.json
   ```
      cd InventoriAppFrontEnd
      npm install
   ```
3. Run Interviewers
   ```
   $ react-native run-android
   ```
